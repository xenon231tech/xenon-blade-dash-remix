# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-06

### Added
- Initial release
- Core game mechanics with slash and dash abilities
- 8 different enemy types with unique behaviors
- Wave-based progression system
- Upgrade system with 8 different upgrades
- Power-up system (Speed, Shield, Double Score)
- Combo system with multiplier bonuses
- Mobile-optimized touch controls
- Boss encounters with special mechanics
- Particle effects and visual feedback
- Score tracking and high score system
- Wave introduction screens
- Pause and resume functionality

### Features
- Fast-paced real-time combat
- Strategic upgrade selection between waves
- Progressive difficulty scaling
- Responsive touch and keyboard controls
- Visual effects and animations
- Audio system integration
- Responsive design for all devices

### Performance
- 60 FPS target on desktop
- 30-60 FPS on mobile devices
- Optimized particle system
- Efficient collision detection
- Low memory footprint
