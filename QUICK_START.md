# Quick Start Guide

## Playing the Game
1. Visit: https://remix.gg/g/9ec88076-bb14-4029-a916-3bbf52929dca
2. Click "TAP TO START"
3. Control with Joystick or Arrow Keys

## Game Basics
- **Objective**: Survive waves of enemies
- **Score**: Defeat enemies and build combos
- **Upgrades**: Choose between 8 upgrades each wave
- **Health**: You have 3 HP (upgradeable)

## Tips & Tricks
1. Build Combos: Kill enemies quickly
2. Choose Upgrades Wisely: Balance offense and defense
3. Keep Moving: Don't stay in one place
4. Use Dash: Escape dangerous situations
5. Manage Health: Heal when possible
