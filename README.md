# Blade Dash Remix

A high-octane action game built with Remix SDK and Canvas. Slash enemies, dodge attacks, and survive increasingly difficult waves!

## Features

- **Fast-paced Combat**: Real-time slash mechanics with responsive controls
- **Wave-based Progression**: Escalating difficulty with boss encounters
- **Upgrade System**: Enhance your abilities between waves
- **Power-ups**: Collect temporary boosts (Speed, Shield, Double Score)
- **Combo System**: Build combos for multiplied rewards
- **Mobile Optimized**: Touch controls with joystick and dash buttons
- **Visual Effects**: Particle systems, screen shake, and smooth animations
- **Audio**: Dynamic sound effects and background music

## Game Mechanics

### Combat
- **Slash**: Tap the attack button to slash nearby enemies
- **Dash**: Use the dash button to evade and reposition
- **Movement**: Use the joystick to move around the arena

### Upgrades
Choose from 8 different upgrades between waves:
- **Vitality**: +1 Max HP
- **Swift Feet**: +15% Speed
- **Sharp Blade**: +1 Slash Damage
- **Wide Arc**: +25% Range
- **Treasure Hunter**: +0.5x Score
- **Quick Dash**: -25% Dash Cooldown
- **Barrier**: Shield for 8 seconds
- **Rejuvenate**: Heal 1 HP

### Enemy Types
- **Normal**: Standard red demon
- **Fast**: Quick fire spirit
- **Tank**: Slow but tanky golem
- **Swarm**: Small imp enemies
- **Shooter**: Ice mage with projectiles
- **Teleport**: Phantom that teleports
- **Splitter**: Slime that splits
- **Boss**: Demon Lord with special attacks

## Controls

### Desktop
- **Arrow Keys / WASD**: Move
- **Space**: Dash
- **Click**: Attack

### Mobile
- **Joystick**: Move character
- **Dash Button**: Perform dash
- **Attack**: Automatic on proximity
