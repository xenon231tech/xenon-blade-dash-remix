# Advanced Strategies

## Optimization Techniques

### Perfect Kiting
- Maintain distance while attacking
- Use dash to reposition
- Attack while moving
- Minimize damage taken

### Combo Mastery
- Never take damage
- Kill enemies efficiently
- Chain kills for multiplier
- Reach 100+ combos

### Upgrade Synergies
- Speed + Dash = Evasion build
- Damage + Range = Offense build
- Vitality + Shield = Tank build
- Score + Treasure = Economy build

## Wave-by-Wave Breakdown

### Waves 1-5: Learning
- Pick any upgrades
- Focus on survival
- Learn enemy patterns
- Build confidence

### Waves 6-10: Specialization
- Choose upgrade path
- Optimize for your style
- Start building combos
- Aim for 0 damage runs

### Waves 11-20: Mastery
- Execute perfect strategy
- Maximize score
- Handle all enemy types
- Compete for records

### Waves 21+: Endless
- Adapt to random spawns
- Maintain composure
- Manage resources
- Push for high scores

## Challenge Runs

### No Damage Run
- Complete waves without taking damage
- Requires perfect execution
- Massive score multiplier
- Ultimate test of skill

### Speed Run
- Complete waves as fast as possible
- Minimize time per wave
- Requires route optimization
- Leaderboard competition

### Specific Upgrade Run
- Use only certain upgrades
- Restrict upgrade choices
- Test different strategies
- Community challenges
