# Mobile Gaming Guide

## Touch Controls

### Joystick
- Left side of screen
- Drag to move
- 8-directional movement
- Analog input supported

### Action Buttons
- Dash button (right side)
- Attack button (right side)
- Pause button (top)

### Gestures
- Tap to attack
- Swipe to dash
- Pinch to zoom (future)
- Shake to pause (future)

## Mobile Optimization

### Screen Size
- Responsive layout
- Adjusts to device size
- Portrait and landscape
- Notch-aware

### Performance
- Optimized for mobile
- Reduced particle count
- Lower resolution option
- Battery saver mode

### Accessibility
- Large touch targets
- High contrast mode
- Text scaling
- Haptic feedback

## Tips for Mobile
- Use landscape for better visibility
- Adjust joystick sensitivity
- Enable haptic feedback
- Close other apps
- Use WiFi for best performance
