# Player Guide

## Getting Started

### First Game
1. Click 'Start Game'
2. Use arrow keys or WASD to move
3. Click to attack nearby enemies
4. Press Space to dash
5. Survive as long as possible

### Understanding the UI
- **Health Bar**: Your remaining health
- **Wave Counter**: Current wave number
- **Score**: Your current score
- **Combo Counter**: Current combo multiplier
- **Upgrades**: Available upgrades between waves

## Basic Strategy

### Early Game (Waves 1-5)
- Focus on learning controls
- Build up health with Vitality upgrades
- Learn enemy patterns
- Don't rush upgrades

### Mid Game (Waves 6-15)
- Balance offense and defense
- Specialize in 2-3 upgrade paths
- Learn kiting techniques
- Build combo chains

### Late Game (Waves 16+)
- Optimize upgrade choices
- Master all enemy patterns
- Execute perfect runs
- Compete for high scores

## Tips and Tricks

### Movement
- Use dash to dodge incoming attacks
- Position yourself between enemies
- Keep moving to avoid getting cornered
- Use arena edges strategically

### Combat
- Attack in groups when possible
- Prioritize dangerous enemies first
- Use terrain to your advantage
- Learn enemy attack patterns

### Upgrades
- Don't always pick damage
- Survivability is key
- Synergize upgrades
- Plan ahead
