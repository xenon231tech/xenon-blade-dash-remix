# Getting Started Tutorial

## Step 1: Launch the Game
1. Visit the game URL
2. Wait for assets to load
3. Click 'Start Game'

## Step 2: Learn Controls
- **Move**: Arrow keys or WASD
- **Attack**: Left click or spacebar
- **Dash**: Space or right click

## Step 3: First Battle
1. Move around the arena
2. Click on enemies to attack
3. Avoid enemy attacks
4. Survive the wave

## Step 4: Choose Upgrades
1. Wave completes
2. Choose 1 of 3 upgrades
3. Upgrades apply immediately
4. Continue to next wave

## Step 5: Game Over
1. Health reaches 0
2. Game ends
3. View final score
4. Start new game
